Coloque junto das respostas como seriam a contagem de pontos pra definir a resposta

Vc pode criar um arquivo json com essas questões, colocadas da seguinte forma:
{
  "title":"Qual personagem de anime você seria?",
  "questions":[
    {
      "id":1,
      "question": "",
      "options":[
        {"id": 1, "name":"", "alias":""},
      ]
    },

Onde alias é a contagem dos pontos.
